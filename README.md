# Turio AI Backend

Backend for the Turio AI application, built with NestJS and TypeScript.

## Description

REST API to manage users, tourist routes, and media content.  
The backend uses:
- NestJS as the main framework
- Firebase Authentication for login (email, Google, Apple)
- PostgreSQL + TypeORM for data persistence
- Azure Blob Storage for file uploads

## Getting started

### Prerequisites

- Node.js (LTS)
- npm
- PostgreSQL database
- Firebase project with Authentication enabled

### Installation

- npm install
